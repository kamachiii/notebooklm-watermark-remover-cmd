# NotebookLM Watermark Remover

A simple, efficient Python tool designed to remove the "NotebookLM" watermark found in the bottom-right corner of PDF slides generated by NotebookLM. 

## How It Works

Instead of cropping or distorting the image, this tool:
1.  **Analyzes** the area immediately surrounding the watermark on each page.
2.  **Detects** the background color by sampling multiple pixels.
3.  **Covers** the watermark with a precisely placed rectangle of the matching background color.

This approach ensures the watermark is hidden cleanly, preserving the look of the slide even if the background color changes from page to page.

## Prerequisites

- **Python 3.6+** installed on your system.
- **pip** (Python package manager).

## Installation

1.  **Clone or Download** this folder to your local machine.
2.  Open a terminal (Command Prompt, PowerShell, or Terminal) and navigate to this folder:
    ```bash
    cd notebooklm-watermark-remover
    ```
3.  (Optional but recommended) Create a virtual environment:
    ```bash
    python -m venv venv
    # Windows:
    venv\Scripts\activate
    # Mac/Linux:
    source venv/bin/activate
    ```
4.  Install the required dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

### Basic Usage
Place your PDF file (e.g., `slides.pdf`) in the same folder or know its path. Run the script providing the input file:

```bash
python remover.py slides.pdf
```

The tool will automatically create a new file named `slides_cleaned.pdf` in the same directory.

### Custom Output Path
You can specify a custom name or location for the output file using the `-o` flag:

```bash
python remover.py my_presentation.pdf -o final_version.pdf
```

### Help
To see all available options:

```bash
python remover.py --help
```

## Troubleshooting

-   **"ModuleNotFoundError: No module named 'fitz'"**: Make sure you ran `pip install -r requirements.txt`. The `fitz` module is part of the `pymupdf` package.
-   **Watermark still visible?**: This tool is optimized for the standard NotebookLM watermark position (bottom-right). If the layout is significantly different, the coordinates in `remover.py` might need adjustment.
